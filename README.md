# Systems Learning

A structured systems engineering learning repository focused on:

- memory models
- execution semantics
- concurrency
- synchronization
- process isolation
- scheduling
- coordination failures
- performance tradeoffs

This repository is not intended to be a collection of tutorial-style examples.

The goal is to understand how real systems execute under:
- shared memory
- concurrent execution
- synchronization constraints
- scheduler interleavings
- ownership/lifetime rules
